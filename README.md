# teTwoCircle

Két kör alakú telek sugarának bekérése,
majd területük számtása és összegzése.

Tesztelésre szánt, JavaScript alkalmazás.

## Gyorsbeállítás

```bash
git clone https://github.com/oktatte/tetwocircle.git
cd tetwocircle
npm install
```
